  SIP URI
    http://wikipedia.org/wiki/SIP_URI_scheme

    本系统使用的 :term:`SIP` :term:`URI` 是 :term:`RFC 3986` 的一个简化子集的变种。
    格式是::

      [[sip:]user@]<host>[:port]

    如::

      sip:13898877785@192.168.5.10:5060
      sip:13898877785@192.168.5.10
      sip:192.168.5.10:5060
      sip:192.168.5.10
      192.168.5.10:5060
      192.168.5.10
      13898877785@192.168.5.10:5060
      13898877785@192.168.5.10
